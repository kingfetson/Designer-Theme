# CreativStudio - Designer Portfolio Theme

A bold, creative portfolio theme with bright gradients, big imagery, smooth animations, and real-time customization.

![Designer Theme Preview](https://via.placeholder.com/800x450?text=CreativStudio+Theme)

## ✨ Theme Features

- **🌈 Bright Gradients** - 5 stunning gradient themes (Sunset, Ocean, Aurora, Candy, Forest)
- **🎨 Big Hero Images** - Full-width cinematic backgrounds with parallax effect
- **✨ Smooth Animations** - Float-in, hover scales, gradient shifts
- **🎭 Creative Layout** - Asymmetric grid, artistic card design
- **📱 Fully Responsive** - Looks stunning on all devices
- **💾 Auto-save** - All changes persist in browser
- **📤 Export/Import** - Backup and restore your portfolio
- **🎯 Project Management** - Add/edit/delete creative projects with images

## 🚀 Quick Start

1. **Download** all files (`index.html`, `style.css`, `script.js`)
2. **Open** `index.html` in your web browser
3. **Start creating** - Edit sidebar controls to see real-time changes
4. **Customize** - Choose gradient themes, add big imagery, showcase projects

## 🎨 Visual Features

### Gradient Themes
| Theme | Colors |
|-------|--------|
| Sunset Blaze | Red → Orange → Pink |
| Ocean Breeze | Teal → Blue → Purple |
| Aurora Dream | Mint → Deep Purple → Violet |
| Candy Burst | Hot Pink → Yellow → Orange |
| Forest Magic | Green → Blue → Dark Teal |

### Animation Effects
- Hero content float-in animation
- Card hover lift with scale
- Smooth gradient transitions
- Icon pulse effects
- Project image zoom on hover
